# WinDots
My public windows dots.

## What is this?
These are my dotfiles for windows.  
These dotfiles use komorebi with the `Scrolling` layout to try and get close to the niri experience on windows.

## installation
1. Clone this repo
   ```
   git clone https://github.com/KingIwrs/.windots.git %userprofile%\.windots
   ```
2. From the directory `%userprofile%\.windots\setup\fonts`, install "`Sono Merged`" and "`Nerd Fonts Symbols`" fonts
   - The other fonts are merged together in "`Sono Merged`"
3. Open settings > search "sudo" > Enable "sudo" > Set to inline
4. Run the script `%userprofile%\.windots\setup\setup.bat`
