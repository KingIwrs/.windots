# WinDots
My public windows dots.

## What is this?
These are my dotfiles for windows. I use WezTerm with nushell, so it installs those f.x. by defualt.  
These dotfiles use komorebi and the scrolling layout to try and get close to the niri experience.

## installation
 - Clone this repo
    ```
    git clone https://github.com/KingIwrs/.windots.git %userprofile%\.windots
    ```
 - From the directory .windots\setup\fonts, install "Sono Merged" and "Nerd Fonts Symbols" fonts
    - The other fonts are merged together in "Sono Merged"
 - Open settings > search "sudo" > Enable "sudo" > Set to inline
 - Run the script ".windots\setup\setup.bat"
